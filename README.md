## 🚀 올리는 법
1. github.com → 새 저장소(Public) 만들기
2. "uploading an existing file" 클릭
3. 위 9개 파일 **전부** 끌어다 놓기
4. Commit changes
5. Settings → Pages → Branch: main → Save
6. 1~2분 뒤 주소 완성!
